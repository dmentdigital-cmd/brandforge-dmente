## 🚀 Como Comenzar

### 1. Clonar Repositorio
```bash
git clone <repo-url>
cd brandforge-dmente
```

### 2. Backend (Primer Terminal)
```bash
cd backend
npm install
cp .env.example .env
# Editar .env con credenciales PostgreSQL
npm run dev
```

### 3. Frontend (Segundo Terminal)
```bash
cd frontend
npm install
npm run dev
# Abre http://localhost:5173
```

### 4. Database (Tercera Terminal)
```bash
psql -U postgres
CREATE DATABASE brandforge_db;
# Backend sincroniza automáticamente
```

### 5. Probar
1. Regístrate con nuevo email
2. Completa Módulo 1
3. Descarga PDF
4. Explora otros módulos
